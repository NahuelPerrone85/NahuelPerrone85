### 🔍 Namaste Yoga — En detalle

> App de reservas de clases de yoga con sistema de pagos real integrado

```
✅ Autenticación completa (Auth.js v5 + JWT + sesiones)
✅ Calendario semanal con disponibilidad en tiempo real
✅ 4 planes de membresía con pago real via Stripe + webhooks
✅ Dashboard personal con historial y estado de membresía
✅ RBAC + middleware de protección de rutas + validación Zod
✅ CI/CD con GitHub Actions + deploy automático en Vercel
✅ Base de datos relacional modelada con Prisma ORM + Supabase
```
